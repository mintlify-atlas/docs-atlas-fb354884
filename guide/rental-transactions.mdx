# Rental Transaction Commissions

Rental transactions in the Tabulador system have a simplified commission structure compared to sales. You receive a fixed percentage of the rental cost.

## How Rental Commissions Work

<Note>
  Rental commissions are **45%** of the net amount after the 7% deduction for taxes and administrative costs.
</Note>

### The Simple Formula

Unlike sale transactions with multiple scenarios, rental commissions follow a single calculation:

<Steps>
  <Step title="Start with Rental Cost">
    The monthly rental cost is your starting point.
    
    **Example:** $18,000 MXN per month
  </Step>

  <Step title="Apply 7% Deduction">
    Deduct 7% for taxes and administrative costs:
    
    ```
    Net Amount = Rental Cost × 0.93
    ```
    
    **Example:** $18,000 × 0.93 = $16,740
  </Step>

  <Step title="Calculate Your 45%">
    You receive 45% of the net amount:
    
    ```
    Your Commission = Net Amount × 0.45
    ```
    
    **Example:** $16,740 × 0.45 = $7,533
  </Step>
</Steps>

## Complete Calculation Example

Let's walk through a real rental transaction:

### Example 1: Standard Rental

<CodeGroup>
```javascript Calculation
// Input
const rentalCost = 18000 // MXN per month

// Step 1: Calculate principal commission
const principalCommission = rentalCost // $18,000

// Step 2: Apply 7% deduction
const netCommission = principalCommission * 0.93 // $16,740

// Step 3: Calculate your 45%
const yourCommission = netCommission * 0.45 // $7,533

console.log(`Your commission: $${yourCommission.toFixed(2)} MXN`)
// Output: Your commission: $7,533.00 MXN
```

```typescript Source Code
// From calculations.ts:37-58
export function getBaseMontos(costo: string, comision: string): BaseMontos {
  const costoNum = Number(costo || 0)
  const comisionNum = Number(comision || 0)
  
  // For rentals
  const comisionPrincipalRenta = costoNum
  const comisionNetaRenta = comisionPrincipalRenta * 0.93
  const montoRenta = comisionNetaRenta * 0.45
  
  return {
    costoNum,
    comisionNum,
    // ... other fields
    comisionPrincipalRenta,
    comisionNetaRenta,
    montoRenta,
  }
}
```
</CodeGroup>

**Breakdown:**
- Rental Cost: **$18,000 MXN**
- Principal Commission: **$18,000** (same as rental cost)
- Net Commission (after 7% deduction): **$16,740**
- **Your Payment: $7,533 MXN** (45% of net)

## Understanding the 7% Deduction

<Note>
  The 7% deduction covers:
  - Taxes (impuestos)
  - Administrative costs
  - Legal fees
  
  This is automatically applied before calculating your commission.
</Note>

The deduction is defined in the system as:
```
Comision neta = Comision principal × 0.93
```

**Code Reference:** `calculations.ts:43`

## When Results Are Displayed

The system only shows results when the rental cost is greater than zero:


## API Integration

If you're integrating with the calculation system programmatically:

```typescript
import { getBaseMontos } from '@/lib/tabulador/calculations'

// For rental transactions
const rentalCost = "18000" // as string
const commission = "0"     // not used for rentals

const result = getBaseMontos(rentalCost, commission)

console.log(result.montoRenta) // Your 45% commission
// Output: 7533
```

**Important Fields:**
- `comisionPrincipalRenta`: Principal commission (equals rental cost)
- `comisionNetaRenta`: Net commission after 7% deduction
- `montoRenta`: Your final 45% commission

## Common Questions

<AccordionGroup>
  <Accordion title="Why is the rental commission always 45%?">
    The 45% rate is a fixed company policy for all rental transactions, regardless of property value or agent involvement. This simplifies calculations and ensures consistent compensation.
  </Accordion>
  
  <Accordion title="Can the percentage increase like in sale transactions?">
    No. Unlike sale transactions where percentages can increase based on your involvement (e.g., 15% → 25%), rental commissions remain fixed at 45%.
  </Accordion>
  
  <Accordion title="Is the 7% deduction negotiable?">
    No. The 7% deduction is mandatory and covers required taxes and administrative costs. It's applied before calculating your 45%.
  </Accordion>
  
  <Accordion title="What if the rental includes utilities or services?">
    Enter only the base rental cost, not including utilities, services, or deposits. The commission is calculated on the monthly rent amount.
  </Accordion>
  
  <Accordion title="Do I need to answer scenario questions for rentals?">
    No. Rental transactions skip all scenario questions (exclusive, listed, sold, etc.). You only need to enter the rental cost.
  </Accordion>
</AccordionGroup>
