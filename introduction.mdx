## Overview

Tabulador de Comisiones Tamivar is a Next.js web application designed for Tamivar real estate advisors to calculate commission distributions based on transaction scenarios. The application provides an interactive questionnaire-based workflow that determines the appropriate commission percentage based on factors like property exclusivity, listing responsibility, sales responsibility, and team collaboration.

## How It Works

The application guides you through a series of questions to determine the correct commission split:

1. **Select Transaction Type**: Choose between property sale or rental
2. **Enter Transaction Details**: Provide the property cost and commission percentage
3. **Answer Scenario Questions**: Respond to questions about exclusivity, listing, and sales responsibility
4. **Review Calculation**: See your commission percentage and final amount after tax deductions
5. **Download Ticket**: Generate a downloadable ticket with the full breakdown

<Note>
All commission calculations automatically include a 7% tax deduction. The final amount displayed is what you will receive after taxes.
</Note>

## Transaction Types

### Property Sales

For property sales, commissions range from 1-10% of the property value (customizable). The final percentage you receive depends on:

- Whether the property is an exclusive listing
- Whether you listed the property
- Whether you sold the property
- If working with external brokers
- If working in a team with other advisors

### Property Rentals

For rental properties, the calculation is simpler:
- You receive 45% of the monthly rental cost
- This is a fixed percentage for all rental transactions
- 7% tax is automatically deducted from this amount
