## Overview

The Tabulador system generates downloadable commission tickets as PNG images. These tickets provide a professional summary of the commission calculation that agents can save, print, or share.

<Note>
Tickets are generated client-side using HTML5 Canvas and downloaded directly to the user's device. No server processing is required.
</Note>

## Ticket Types

The system generates two types of tickets based on operation type:

## Image Generation Technical Details

The ticket image is generated using HTML5 Canvas in `lib/tabulador/ticket-image.ts`.

### Canvas Configuration

From `lib/tabulador/ticket-image.ts:21-52`:

```typescript
export function downloadTicketAsImage(ticketTexto: string) {
  if (!ticketTexto) return

  const canvas = document.createElement("canvas")
  const ctx = canvas.getContext("2d")
  if (!ctx) return

  // Canvas settings
  const width = 1200
  const paddingX = 64
  const paddingY = 72
  const lineHeight = 42
  const maxTextWidth = width - paddingX * 2

  // Font for text wrapping measurement
  ctx.font = "500 32px Arial"
  
  // Wrap text lines
  const rawLines = ticketTexto.split("\n")
  const wrappedLines: string[] = []
  rawLines.forEach((line) => {
    wrapLine(ctx, line, maxTextWidth).forEach((wrapped) => wrappedLines.push(wrapped))
  })

  // Calculate dynamic height
  const height = Math.max(720, paddingY * 2 + wrappedLines.length * lineHeight)
  canvas.width = width
  canvas.height = height
}
```

### Visual Design

The ticket has a three-layer design:

```typescript
// Layer 1: White background
ctx.fillStyle = "#ffffff"
ctx.fillRect(0, 0, width, height)

// Layer 2: Light gray border area
ctx.fillStyle = "#e2e8f0"
ctx.fillRect(24, 24, width - 48, height - 48)

// Layer 3: White content area
ctx.fillStyle = "#ffffff"
ctx.fillRect(32, 32, width - 64, height - 64)
```

### Text Rendering

From `lib/tabulador/ticket-image.ts:54-72`:

```typescript
let y = paddingY
wrappedLines.forEach((line, index) => {
  const isTitle = index === 0 && line.trim().toUpperCase() === "DESGLOSE"
  
  if (isTitle) {
    ctx.font = "700 44px Arial"  // Bold 44px for title
    ctx.fillStyle = "#0f172a"     // Dark slate color
  } else {
    ctx.font = "500 32px Arial"  // Medium 32px for body
    ctx.fillStyle = "#1e293b"     // Medium slate color
  }

  if (!line.trim()) {
    y += lineHeight  // Empty line spacing
    return
  }

  ctx.fillText(line, paddingX, y)
  y += lineHeight
})
```

### Text Wrapping

Long lines are automatically wrapped to fit the canvas width:

```typescript
function wrapLine(ctx: CanvasRenderingContext2D, text: string, maxWidth: number) {
  if (!text.trim()) return [""]
  
  const words = text.split(" ")
  const lines: string[] = []
  let current = words[0] ?? ""

  for (let i = 1; i < words.length; i += 1) {
    const candidate = `${current} ${words[i]}`
    if (ctx.measureText(candidate).width <= maxWidth) {
      current = candidate
    } else {
      lines.push(current)
      current = words[i]
    }
  }

  lines.push(current)
  return lines
}
```

## Currency Formatting

All monetary values in tickets use Mexican Peso formatting from `lib/tabulador/format.ts`:

```typescript
export function formatMXN(value: number) {
  return new Intl.NumberFormat("es-MX", {
    style: "currency",
    currency: "MXN",
    maximumFractionDigits: 2,
  }).format(value)
}
```

**Output examples:**
- `formatMXN(18000)` → `$18,000.00`
- `formatMXN(91140)` → `$91,140.00`
- `formatMXN(7533.50)` → `$7,533.50`

## Tax Disclaimer

All tickets include the tax disclaimer from `lib/tabulador/constants.ts`:

```typescript
LEYENDA_IMPUESTOS = "*Este monto ya tiene el 7% de impuestos descontado"
```

This reminds users that the displayed amounts have already had the 7% tax deducted.

<Warning>
The 7% tax deduction is applied during calculation, NOT during ticket generation. The ticket displays the final amounts after tax has already been deducted.
</Warning>

## Common Issues

<AccordionGroup>
  <Accordion title="Ticket button is disabled or not visible">
    **Cause:** The calculation is incomplete or invalid.
    
    **Solution:** 
    - For rentals: Enter a valid rental cost greater than 0
    - For sales: Enter property price, commission %, and answer all scenario questions
    
    The button only appears when `puedeVerResultado` or `puedeVerResultadoRenta` returns `true`.
  </Accordion>

  <Accordion title="Downloaded file has no content">
    **Cause:** The `ticketTexto` variable is empty or the Canvas API failed.
    
    **Solution:**
    - Check browser console for errors
    - Ensure you're using a modern browser with Canvas support
    - Try refreshing the page and recalculating
  </Accordion>

  <Accordion title="Text is cut off or wrapped incorrectly">
    **Cause:** Very long words or numbers exceed the maximum text width.
    
    **Solution:**
    The `wrapLine()` function handles word wrapping automatically. If text still appears cut off, this may be a browser-specific rendering issue. Try a different browser.
  </Accordion>
</AccordionGroup>
